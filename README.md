# edsonluizcandido

repositorio para meu site pessoal www.edsonluizcandido.com.br

Site escrito em Blazor Server para geração de conteudo dinâmico, a partir da atualização do arquivo '/Pages/Index.md'.

## para devs

	- utilizado a biblioteca Markdig para converter markdown em html
	- ASP.NET MVC programado em C#
	- docker para subir o site em um container e rodar em servidor linux
